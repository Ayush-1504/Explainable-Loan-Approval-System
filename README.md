# 💰 Explainable Loan Approval System

An end-to-end Machine Learning web application that predicts loan approval status using applicant details.

---

# 🚀 Live Demo

🔗 Streamlit App: https://your-streamlit-link.streamlit.app

🔗 GitHub Repository: https://github.com/Ayush-1504/Explainable-Loan-Approval-System

---

# 📌 Features

- Loan approval prediction using Machine Learning
- Logistic Regression, Random Forest, and XGBoost models
- Explainable AI using SHAP
- Feature importance visualization
- Interactive Streamlit web application
- Real-time prediction confidence score
- User-friendly interface
- End-to-end ML pipeline

---

# 🧠 Machine Learning Models Used

| Model | Accuracy |
